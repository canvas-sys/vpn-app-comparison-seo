## FAQ

**Q: Does a VPN automatically make me anonymous?**  
No. A VPN helps with privacy and security focus (traffic encryption, IP masking), but accounts, cookies, fingerprinting, and logged-in services still identify you.

**Q: What should I test first after install?**  
Test the **Kill Switch**. Disconnect your network mid-session and confirm traffic stops, then reconnect and verify the tunnel restores cleanly.

**Q: Why do speeds vary so much between servers?**  
Distance, load, and routing. Pick nearby **Global Servers** first, then try alternates if latency spikes.

**Q: Is AES-256 the only thing that matters for security?**  
No. **AES-256 encryption** is table stakes. Protocol quality, DNS handling, leak protection, and defaults matter just as much.
